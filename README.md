# Acesso ao Dion

Este projeto é uma forma alternativa de acessar as notas do Dion da Universidade Regional de Blumenau utilizando Selenium.

## WSDion

Versão Web Service da ferramenta, para que possa ser utilizada por outros projetos

## CLIDion

Versão em linha de comando, para um acesso mais rápido às notas do dion, sem precisar acessar através do navegador.
